# 🎮 Pixel Wars Podcast

Proyecto del módulo M9. Un podcast gamer donde debatimos sobre videojuegos.
El primer episodio es sobre el debate eterno: PC vs Consola.

🌐 **Web:** https://pixel-wars-podcast.vercel.app

---

## Cómo hice el podcast

Primero pensé el tema y cómo estructurar el episodio: introducción, argumentos de cada lado, mi opinión y cierre. Luego grabé la voz con Audacity, quité el ruido de fondo y corté los silencios que sobraban. Añadí música de fondo y usé Auto Duck para que bajara cuando hablo. Al final lo exporté a MP3 con FFmpeg para que no pesara demasiado.

Para la web usé React con Vite y Tailwind CSS. La subí a GitHub y la desplegué en Vercel.

---

## Herramientas que usé

- **Audacity** – para grabar y editar el audio
- **FFmpeg** – para optimizar el MP3
- **React + Vite** – para la web
- **Tailwind CSS** – para el diseño
- **Vercel** – para publicarla online
